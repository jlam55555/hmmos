Detailed Description

template<Base com>
class serial::Serial< com >

A very simple abstraction of a serial device.

This is templated by base port address so that the port computations can be constexpr.

Although if we really cared about performance then read() and write() would be in pure asm and would be defined in the header so that they are inlinable.

Since this is stateless (the only state being the base port address) this class is functionally equivalent to a set of templated functions.
