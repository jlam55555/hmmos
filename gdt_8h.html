◆ init()

void arch::gdt::init ( )

Re-init the GDT.

We need to reconfigure this as the original GDT lies in bootloader-reclaimable memory. This also includes the TSS and ring 3 segments which were not present in the bootloader GDT (and also doesn't include the unreal mode segment.

Memory segments: GDT[0] = null descriptor GDT[1] = ring 0 code GDT[2] = ring 0 data GDT[3] = ring 3 code GDT[4] = ring 3 data GDT[5] = TSS
