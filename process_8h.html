Detailed Description

Process abstraction (a.k.a. userspace thread).

Syscalls are implemented as methods of Process.

The process of creating a process (via fork/exec):

  1. Map memory from executable.
  2. Setup standard FDs.
  3. Setup kernel thread (which simply jumps to userspace start label).
  4. Add thread to scheduler.

Syscalls are implemented as methods of Process.
