void arch::sched::switch_stack ( void **  old_stk,
void *  new_stk 
)

Switch stacks. This saves the old tasks's state (callee-save registers) on its stack, and restores the new task's state from its stack.

new_stk should either be set up from a previous call to switch_stack() or from setup_stack(). For this reason, it is not safe for new_stk to be the currently running thread.
