Detailed Description

Very simple printing utilities via BIOS text screen memory.

The bootloader uses these directly, while the kernel has more advanced printf-like functions. In the interest of not including the whole printf* functionality to the bootloader, this is the only shared logic.

These technically should be usable from (un)real and protected mode, but these are only currently used in protected mode. The bootloader code contains very simple real-mode printing logic.
