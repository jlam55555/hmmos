Detailed Description

Simple round-robin allocator that doesn't use auxiliary memory.

This keeps track of used pages via the PFT and uses a simple round-robin linear scan to find free memory regions.

N.B. The needle does not get incremented after allocation. This means that in the case of an alloc-free-alloc pattern, the second alloc will return the same base address as the first alloc if the second alloc is of no greater size than the first.
