◆ pt_setup()

bool pt_setup ( void *  kernel_paddr)

Allocate space for page table mapping:

  • 1 page directory (will use 4MB pages for HHDM).
  • 1 page table for 1MB direct map (can't use 4MB page since we don't want to map the null page).

This can probably fit within the text region of the bootloader (since we have ~32KB) but we may need to allocate more pages in the future, e.g. if using 3-level (PAE) paging. Let's leave this dynamically allocated for now.
