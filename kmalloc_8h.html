Detailed Description

Very simple sequential allocator.

Temporary helper functions before we have a slab allocator. Never frees memory. Maybe we can keep this around (or some version of a sequential allocator w/ vmalloc, so the entire arena is contiguous and can be freed at once) as a very simple allocator model.
