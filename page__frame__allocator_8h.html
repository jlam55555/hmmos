Detailed Description

Allocator for (4KB) physical page frames.

Page frame allocators provide a simple alloc() and free() interface for physical pages. Each PFA is responsible for some contiguous range of memory – it will determine which pages are usable by looking at the PFT's usable regions.

SimplePFA is a very simple round-robin, first-fit allocator that uses the PFT to determine if a page is free or not. This is pretty terrible (first-fit means poor larger allocations, and using a sparse table is bad cache-wise).
